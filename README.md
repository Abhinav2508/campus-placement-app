# Campus Placement App 📱🎓

An AI-ready Campus Placement Management System built using **Flutter + Django REST Framework + JWT Authentication**.

This application digitizes the complete college placement workflow for students and placement coordinators.

---

## 🚀 Features

### 👨‍🎓 Student Side

* Secure Login & Authentication (JWT)
* Profile Management
* Resume Upload
* View Upcoming Companies
* Eligibility Based Filtering (CGPA + Skills)
* Apply to Companies
* Track Application Status Timeline

### 🏢 Placement/Admin Side

* Add Companies
* Define Eligibility Criteria
* Set Deadlines
* Shortlist Students
* Update Interview Status
* Publish Final Results

---

## 🧠 Smart Logic Implemented

* Skill matching between student & company
* Automatic eligibility filtering
* Unique application protection (no duplicate apply)
* Application lifecycle tracking

---

## 🏗️ Tech Stack

**Frontend**

* Flutter
* REST API Integration
* Persistent Login (SharedPreferences)

**Backend**

* Django
* Django REST Framework
* JWT Authentication
* SQLite (development)

---

## ⚙️ How to Run

### Backend

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend

```bash
cd frontend
flutter pub get
flutter run
```
